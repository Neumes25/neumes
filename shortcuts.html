Neumes - using the keyboard shortcuts



For the most commonly used notes, the keyboard can be used for input, as it is much faster. In order to add notes with the keyboard, you have to click on the page, which will become highlighted and take the keyboard inputs. If you click outside of the Virtual Page, it will no longer be highlighted and keystrokes will be ignired. Ths is done in order to be able to use the keyboard for additional tasks, such as writing the text under the notes, or adding text to the page.
